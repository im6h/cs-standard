# Lấy thông tin mạng

MAC:

```
Cmd + <space> term <enter> ifconfig <enter>
```

Window:

```
Win + R <enter> cmd <enter> ipconfig <enter>
```



# PORT

MAC:

Lấy process chiếm cổng <port>
```
lsof -i :<port>
```
Kill process
```
kill -9 <processid>
```
